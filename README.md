# ExcelSearchApp
A standalone Java application that can search <b>exact & similar</b> matching tuples in a single Excel file. Extensions supported are .xls &amp; .xlsx both.



<h3>Steps to install and run</h3>
<li>Clone the repository</li>
<li>Ensure that javafx lib are installed in target system</li>
<li>Change the path to the javafx libs installed in the target system into the runMySearchApp.bat file</li>
<li>save changes and double click. the application should run.</li>


<h4>System pre-requisites</h4>
<li>Java jdk 24 or higher</li>
<li>JavaFX 24.0.2</li>
